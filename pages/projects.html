CPU Simulator
Fall 2021

Along with Computer Architecture Course (ECE M116C), I designed a 5 pipeline staged RISC-V processor simulator.
This simulator is written in C++ and can run any number of instructions given a text file with 32 bit binary numbers in each line!
(If you choose to use 10 specified RISC-V ISA instructions)
In addition to register file, pipeline register, arithmetic logic unit, and controller, I implemented Direct-Mapped and 4-way Set Associative Memory Models for L1 cache.
This simulator emulates the behavior of processor and helped me compare different cache models with different policies.
